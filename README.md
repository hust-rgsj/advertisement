## 项目特性

互联网服务提供企业（例如，搜索服务提供商，翻译服务提供商）通过展示企业客户广告获得服务收益，企业客户向互联网服务提供商购买广告，购买的广告展示在服务使用界面上（Web应用、移动APP、微信小程序等）展示。
