# 互联网广告系统

## 1 企业管理员

### 1.1 管理用户账号权限

#### 1.1.1 基本信息

> 请求路径：admin/customer/status
>
> 请求方式：POST
>
> 接口描述：该接口用于管理用户账号权限

#### 1.1.2 请求参数

参数格式: query参数

| 参数名称   | 类型     | 是否必须 | 备注                |
|--------|--------|------|-------------------|
| status | number | 必须   | 用户账号权限, 0-禁用，1-正常 | 
| id     | number | 必须   | 用户id              |

请求数据样例：

```json
  {
    "status": 0,
    "id": 1
}
```

#### 1.1.3 响应数据

参数格式：application/json

参数说明：

| 参数名称 | 类型     | 是否必须 | 备注  |
|------|--------|------|-----|
| code | number | 必须   |     | 响应码, 200 成功 ; 500 失败 |      |
| msg  | string | 非必须  |     | 提示信息                 |      |
| data | object | 非必须  |     | 返回的数据                |      |

响应数据样例：

```json
{
  "code": 200,
  "msg": null,
  "data": "已解封该账号"
}
```

```json
{
  "code": 200,
  "msg": null,
  "data": "已成功禁用该账号"
}
```
